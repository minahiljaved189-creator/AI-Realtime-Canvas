# AI Real-Time Canvas

## Project Description
This project converts user prompts into real-time canvas shapes using AI.

## Tech Stack
- React + TypeScript
- Node.js + Socket.io
- Groq/OpenAI API
- React Konva

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Open App
http://localhost:5173

## Features
- Prompt to JSON conversion
- Real-time canvas sync
- Draggable shapes
